# 📖 用語集（Glossary）

> Construction-SIEM-Platform で使用される主要な用語・略語の定義集

---

## 🔒 セキュリティ全般

| 用語 | 正式名称 | 定義 |
|------|----------|------|
| **SIEM** | Security Information and Event Management | セキュリティ情報およびイベント管理。ログ収集・相関分析・アラート生成を統合的に行うシステム |
| **UEBA** | User Entity Behavior Analytics | ユーザーおよびエンティティの行動分析。通常パターンからの逸脱を検知し、内部脅威を発見する技術 |
| **IoC** | Indicators of Compromise | 侵害指標。マルウェアハッシュ、悪意あるIP/ドメインなど、攻撃の痕跡を示す情報 |
| **MITRE ATT&CK** | — | 実世界の観察に基づく攻撃手法の分類フレームワーク。戦術（Tactic）と技術（Technique）で構成される |
| **キルチェーン** | Cyber Kill Chain | 偵察→武器化→配送→攻撃→インストール→C2→目的達成の7段階で攻撃プロセスを表現するモデル |
| **RBAC** | Role-Based Access Control | ロールベースアクセス制御。ユーザーにロールを割り当て、ロールに権限を紐付ける認可モデル |

---

## 📊 メトリクス・SLA

| 用語 | 正式名称 | 定義 |
|------|----------|------|
| **MTTD** | Mean Time To Detect | 平均検知時間。脅威が発生してから検知されるまでの平均所要時間 |
| **MTTR** | Mean Time To Respond | 平均対応時間。脅威を検知してから対応が完了するまでの平均所要時間 |
| **SLA** | Service Level Agreement | サービスレベル合意。サービスの品質・可用性に関する合意事項 |
| **EPS** | Events Per Second | 1秒あたりのイベント処理数。システムのスループット指標 |
| **KPI** | Key Performance Indicator | 重要業績評価指標。システムの目標達成度を定量的に測定する指標 |

---

## 📝 ログ・データ形式

| 用語 | 正式名称 | 定義 |
|------|----------|------|
| **CLF** | Common Log Format | 本プロジェクトで定義する正規化されたセキュリティイベント形式。各種ログソースのイベントを統一フォーマットに変換する |
| **CEF** | Common Event Format | ArcSight が定義する標準イベント形式。外部システムとの連携時に使用される |
| **ILM** | Index Lifecycle Management | Elasticsearch のインデックスライフサイクル管理。データの保持・移行・削除ポリシーを自動制御する仕組み |

---

## 🏗 建設現場固有

| 用語 | 定義 |
|------|------|
| **CAD** | Computer-Aided Design。設計図面の作成・編集に使用するソフトウェア |
| **BIM** | Building Information Modeling。建物の3Dモデルに情報を付与した統合データモデル |
| **IoT エッジデバイス** | 現場に設置されるセンサー・カメラ・制御装置などのインターネット接続機器 |
| **オフライン現場** | ネットワーク接続が断続的な建設現場。ローカルバッファリングでイベントを蓄積し、接続復帰時に送信する |

---

## 🛠 技術コンポーネント

| 用語 | 定義 |
|------|------|
| **FastAPI** | Python 製の高性能 Web API フレームワーク。非同期処理・自動ドキュメント生成に対応 |
| **Elasticsearch** | 分散型全文検索・分析エンジン。ログデータのストレージおよびクエリ処理を担当 |
| **Kafka** | 分散メッセージブローカー。ログ収集からストレージまでのイベントパイプラインを担う |
| **Fluentd** | ログ収集・転送エージェント。各種ログソースからイベントを収集し Kafka へ転送する |
| **Grafana** | メトリクス・KPI の可視化ダッシュボード。Prometheus と連携しシステム監視を行う |
| **Kibana** | Elasticsearch のデータ可視化ツール。ログ検索・ダッシュボード作成に使用 |
| **Prometheus** | 時系列データベース。システムメトリクスの収集・保存を行う |
| **Docker Compose** | 複数コンテナのオーケストレーションツール。開発・本番環境の構築に使用 |

---

## 📜 準拠規格

| 用語 | 定義 |
|------|------|
| **ISO 27001** | 情報セキュリティマネジメントシステム（ISMS）の国際規格 |
| **ISO 27001 A.8.15** | イベントログ取得に関する管理策 |
| **ISO 27001 A.8.16** | 監視活動に関する管理策 |
| **NIST CSF 2.0** | 米国 NIST が策定するサイバーセキュリティフレームワーク 2.0 |
| **NIST CSF DE.CM** | DETECT 機能の継続的モニタリングカテゴリ |
| **ISO 20000** | IT サービスマネジメントの国際規格 |
