## 🎯 目的

**建設現場サイバーセキュリティ監視・SIEM統合システム**

建設現場で運用される IoT 機器・CAD/BIM 環境から発生するセキュリティイベントを集約し、リアルタイムで異常を検知・対応するための統合 SIEM プラットフォームを提供する。

---

## 📜 準拠規格

| 規格 | 対応領域 |
|------|----------|
| 🔒 **ISO 27001 A.8.15** | イベントログ取得 |
| 🔒 **ISO 27001 A.8.16** | 監視活動 |
| 🛡 **NIST CSF 2.0 DETECT DE.CM** | 継続的モニタリング |
| ⚙️ **ISO 20000** | IT サービスマネジメント |

---

## 🏢 対象領域

- 🏗 現場 IoT 機器（センサー、カメラ、エッジデバイス）のセキュリティイベント集約
- 📐 CAD/BIM 環境におけるファイルアクセス・エクスポート監視
- 🌐 ネットワーク境界でのトラフィック異常検知
- 👤 ユーザー行動分析（UEBA）による内部不正検知
- 🔗 脅威インテリジェンス（IoC）照合による既知攻撃検出
