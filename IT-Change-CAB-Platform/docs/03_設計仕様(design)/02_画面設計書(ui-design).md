## 1. 画面設計方針

### 1.1 デザインガイドライン

| 項目 | 方針 |
|------|------|
| UIフレームワーク | Ant Design 5.x |
| デザインシステム | Ant Design tokens + カスタムテーマ |
| レイアウト | 固定サイドバー + ヘッダー + コンテンツ領域 |
| レスポンシブ | デスクトップ優先（1366px以上フル対応） |
| カラーテーマ | ライトモード（ダークモード対応予定） |
| アイコン | Ant Design Icons |
| フォント | Noto Sans JP / system-ui |

### 1.2 共通UI要素

| 要素 | 説明 |
|------|------|
| パンくずリスト | 全画面でページ階層を表示 |
| ローディングスピナー | API通信中のローディング表示 |
| エラーメッセージ | フォーム直下・トースト通知での表示 |
| 確認ダイアログ | 状態遷移操作時の確認ダイアログ |
| 成功通知 | 操作成功時のトースト通知（右上表示・3秒自動消去） |
